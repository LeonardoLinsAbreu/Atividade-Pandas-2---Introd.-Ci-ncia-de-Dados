Este repositório contém a resolução da **Atividade 2**, focada em técnicas de limpeza, transformação e agrupamento de dados utilizando a biblioteca Pandas em ambiente Python.

# Tecnologias Utilizadas:

* Python;
* Pandas;
* Jupyter Notebook / VS Code;
* Dataset no formato Parquet.

# Objetivos da Atividade:

O notebook aborda os seguintes processos de análise de dados:

1. **Filtros Avançados:** Filtragem por ano (2022), região (Norte) e valores específicos;
2. **Limpeza de Dados:** Tratamento de valores ausentes (NaN) e substituição de strings;
3. **Normalização:** Conversão de tipos de dados (strings para floats) e padronização de texto (Uppercase);
4. **Tratamento de Duplicatas:** Identificação e remoção de registros repetidos;
5. **Agrupamentos (Groupby):** Cálculos estatísticos por Região e Função Orçamentária.
